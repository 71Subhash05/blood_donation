# Blood Donation App - Complete Project

This repository contains the complete Blood Donation App with proper folder structure.

## Project Structure

```
blood_donation/
├── lib/                    # Flutter source code
│   ├── constants/         # App constants
│   ├── models/           # Data models
│   ├── services/         # API services
│   ├── utils/            # Utility functions
│   └── *.dart           # Main UI pages
├── backend/              # Node.js backend
│   ├── models/          # MongoDB models
│   ├── routes/          # API routes
│   └── server.js        # Main server file
├── android/             # Android build configuration
├── web/                 # Web assets
└── README.md           # Project documentation
```

## Features Implemented

### ✅ Styled Pages:
- **Home Screen** - Full gradient background with hover effects
- **Login Page** - Modern dark theme with gradient cards
- **Signup Page** - 3D animated form with blue theme

### ✅ Functional Pages:
- **Donor Form** - Complete registration with validation
- **Search Page** - Blood donor search with filters

### ✅ Backend:
- **Node.js Server** - Express.js with MongoDB
- **REST API** - Authentication and donor management
- **Database Models** - User and Donor schemas
